# dotfiles

## install

### clone repository
```
git clone https://github.com/kamaboko123/dotfiles.git
```

### nvim
```
ln -s dotfiles/.config/nvim/init.vim ~/.config/nvim/init.vim 
```

### tmux
```
ln -s dotfiles/.tmux.conf ~/.tmux.conf
```

### git
```
ln -s dotfiles/.gitconfig ~/.gitconfig
```
